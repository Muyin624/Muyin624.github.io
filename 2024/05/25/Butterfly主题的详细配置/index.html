hexo详细配置(Butterfly主题)

发表于2024-05-25|学习
|字数总计:1.6k|阅读时长:6分钟|阅读量:

站点配置文件

/网站目录/_config.yml(修改网站各种资料，例如标题、副标题和邮箱等个人资料)

Site 网站

参数 描述
title 网站标题
subtitle 网站副标题
description 网站描述。主要用于SEO，告诉搜索引擎一个关于站点的简单描述，通常包含您网站的关键词
keywords 网站的关键词。支持多个关键词。
author 您的名字
language 网站使用的语言。对于简体中文用户来说，使用不同的主题可能需要设置成不同的值，请参考你的主题的文档自行设置，常见的有 zh-Hans 和 zh-CN。
timezone 网站时区。Hexo 默认使用您电脑的时区。请参考 时区列表 进行设置，如 America/New_York, Japan, 和 UTC 。一般的，对于中国大陆地区可以使用 Asia/Shanghai。

URL（网址）

参数 描述 默认值
url 网址, 必须以http://或https://开头
root 网站根目录 url’s pathname
permalink 文章的 永久链接 格式 :year/:month/:day/:title/
permalink_defaults 永久链接中各部分的默认值
pretty_urls 改写permalink 的值来美化 URL
pretty_urls.trailing_index 是否在永久链接中保留尾部的index.html true
pretty_urls.trailing_html 是否在永久链接中保留尾部的.html(对尾部的 index.html 无效) true

如果您的站点位于子目录中，请将 URL 设置为‘http://yoursite.com/child’并将根目录设置为‘/Child/’

Directory（目录*熟悉hexo后再来修改）

参数 描述 默认值
source_dir 资源文件夹，这个文件夹用来存放内容。 source
public_dir 公共文件夹，这个文件夹用于存放生成的站点文件。 public
tag_dir 标签文件夹 tags
archive_dir 归档文件夹 archives
category_dir 分类文件夹 categories
code_dir Include code 文件夹，source_dir 下的子目录 downloads/code
i18n_dir 国际化（i18n）文件夹 :lang
skip_render 跳过指定文件的渲染。匹配到的文件将会被不做改动地复制到 public 目录中。您可使用 glob 表达式来匹配路径。

Writing（文章）

参数 描述 默认值
new_post_name 新文章的文件名称 :title.md
default_layout 预设布局 post
auto_spacing 在中文和英文之间加入空格 FALSE
titlecase 把标题转换为 title case FALSE
external_link 在新标签中打开链接 TRUE
external_link.enable 在新标签中打开链接 TRUE
external_link.field 对整个网站（site）生效或仅对文章（post）生效 site
external_link.exclude 需要排除的域名。主域名和子域名如 www 需分别配置 []
filename_case 把文件名称转换为 (1) 小写或 (2) 大写 0
render_drafts 显示草稿 FALSE
post_asset_folder 启用 资源文件夹 FALSE
relative_link 把链接改为与根目录的相对位址 FALSE
future 显示未来的文章 TRUE
syntax_highlighter 代码块的设置, 请参考 代码高亮 进行设置 highlight.js
highlight 代码块的设置, 请参考 Highlight.js 进行设置
prismjs 代码块的设置, 请参考 PrismJS 进行设置

Category & Tag（分类和标签）

参数 描述 默认值
default_category 新文章的文件名称 uncategorized
category_map 预设布局
tag_map 在中文和英文之间加入空格

Date / Time format(日期 / 时间格式)

Hexo 使用 Moment.js 来解析和显示时间

参数 描述 默认值
per_page 每页显示的文章量 (0 = 关闭分页功能) 10
pagination_dir 分页目录 page
updated_option 当 Front Matter 中没有指定 updated 时 updated 的取值 mtime

Pagination(分页)

参数 描述 默认值
per_page 每页显示的文章量 (0 = 关闭分页功能) 10
pagination_dir 分页目录 page

Extensions(扩展)

插件: https://hexo.io/plugins/
主题: https://hexo.io/themes/

参数 描述
theme 当前主题名称。值为false时禁用主题
theme_config 主题的配置文件。在这里放置的配置会覆盖主题目录下的 _config.yml 中的配置
deploy 部署部分的设置
meta_generator Meta generator 标签。 值为 false 时 Hexo 不会在头部插入该标签

Butterfly主题配置文件

/网站目录/_config.butterfly.yml/(修改主题内容，如果没有根据下文进行创建)

准备工作

按照主题作者所说，最好在Hexo的根目录创建一个文件_config.butterfly.yml，并把主题目录的_config.yml内容复制到_config.butterfly.yml里，以后只需要在_config.butterfly.yml进行配置。

Hexo会自动合并主题中的_config.yml和_config.butterfly.yml里的配置，如果存在同名配置，会使用_config.butterfly.yml的配置，其优先度较高。

导航栏相关

导航栏设置

参数 描述 默认值
logo 网站的 logo支持图片
display_title 是否显示网站标题 true
fixed 是否固定状态栏 false

菜单目录设置

代码相关

代码行

参数 描述 默认值
highlight_theme 代码高亮主题 darker darker / pale night / light / ocean / mac / mac light
highlight_copy 是否显示复制按钮 true
highlight_lang show the code language 展示代码块语言 true
highlight_shrink true/false 默认折叠/展开代码块内容 / false
highlight_height_limit 代码高度限制，超出的部分会隐藏，并显示展开按钮 false
code_word_wrap 代码换行，是否关闭滚动条 false

图片相关

Social Settings (社交图标設置)

图标名 : url 描述 颜色
fab fa-github : https://github.com/Muyin624/Muyin624.github.io Github black
fas fa-envelope : mailto:muyin624@gmail.com Email
fa-brands fa-qq : fa-brands fa-qq: https://wx.mail.qq.com 3389609979

Butterfly支持font-awesome v6 图标,图标名可以在font-awesome中找到

Favicon（网站图标）

参数 描述 默认值
favicon 图片路径 /img/favicon.png

Avatar (头像)

参数 描述 默认值
img 图片路径 /img/avatar.jpg
effect 是否开启头像的动效 false

顶部图

参数 解释
index_img 主页的 top_img
default_top_img 默认的 top_img，当页面的 top_img 没有配置时，会显示 default_top_img
archive_img 归档页面的 top_img
tag_img tag子页面 的 默认 top_img
tag_per_img tag子页面的 top_img，可配置每个 tag 的 top_img
category_img category 子页面 的 默认 top_img
category_per_img category 子页面的 top_img，可配置每个 category 的 top_img
