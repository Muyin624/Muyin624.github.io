SVG 基本语法

画布

绘制简单图形

矩形（Rectangles）

使用 元素绘制矩形，可以指定矩形的位置、大小、圆角等属性

圆形（Circles）

使用 元素绘制圆形，可以指定圆心坐标和半径

椭圆（Ellipses）

使用 元素绘制椭圆，可以指定椭圆的中心坐标和长短轴的半径

直线（Lines）

使用 元素绘制直线，需要指定起点和终点坐标

多边形（Polygons）

使用 元素绘制多边形，需要指定多个顶点的坐标

折线（Polylines）

使用 元素绘制折线，需要指定多个点的坐标

路径（Paths）

使用 元素绘制路径，可以通过指定一系列的路径命令来绘制各种形状

渐变和填充

使用 或 定义渐变
使用 fill 和 stroke 属性指定填充和描边样式

文本和字体

使用 元素插入文本
使用 font-family、font-size 等属性控制文本样式

动画和交互

使用CSS或JavaScript创建动画效果。
添加事件处理器实现交互功能，如鼠标点击、悬停等。
