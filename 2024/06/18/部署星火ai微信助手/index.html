部署星火ai微信助手

发表于2024-06-18|学习
|字数总计:360|阅读时长:1分钟|阅读量:

前言

注：必须并且只要有自己的一个域名（阿里云等平台新用户一元即可）

基于vercel的serverless服务，把星火认知大模型接入微信公众号
1、用户通过个人公众号，输入对话内容
2、通过公众号服务器配置地址进行请求访问，此处的地址需要使用自己的个人域名
3、通过地址请求到使用vercel.com
4、最终请求到讯飞星火模型接口

部署流程(申请大模型,公众号需自行搜索)

注册星火认知大模型，创建一个应用

创建完成后进入服务管理页面

查找 APPID，APISecret，APIKey

购买域名(举例阿里云)

我的是.xyz,9元一年

微信公众平台设置

进入微信公众平台

url

token

TOKEN是自定义的，随便编一个，但需要与下文中的TOKEN保持一致

EncodingAESKey，为前面项目配置中设置的参数

点击后面的随机生成即可

消息加解密方式

选择明文模式

图示

先不要提交，提交会校验TOKEN，接下来我们才会开始部署

Vercel 部署

以个人开发者名义进行注册并登录

创建项目

引入大佬开源的 Github 项目地址

我用的项目名称是vercel-wxxinghuo-bot

等待一会~
